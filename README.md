#### Subscribe to this repository
``` console
user@termux:~$ apt-get update && apt-get upgrade -y
user@termux:~$ apt-get install wget
user@termux:~$ wget https://rendiix.github.io/install-repo.sh
user@termux:~$ bash install-repo.sh
```
